# Backend Setup

we can need to install some dependencies. let's install them.

first dependencies we need to install, are:

- TypeORM
- Postgres Driver
- Config Module
- JWT Module
- Passport Module
- Dotenv Module
- Validation Module

let's install them.

```bash
npm install --save @nestjs/typeorm typeorm pg @nestjs/config @nestjs/jwt @nestjs/passport dotenv class-validator class-transformer
```
